# BacktestingEV

## Presentation

**Development of a Back-Testing Procedure in Julia**

This package will allow you to backtest your forecasting model.


## Installation

```
pkg> add https://github.com/AMauro130/BacktestingEV.jl.git
```
Then
```
using BacktestingEV
```
